# theTenSpot

Curated Amazon shopping app that takes the guessing out of buying quality products.


## Getting Started

> git clone https://github.com/etchoi/theTenSpot.git  
> cd theTenSpot
> npm install  
> npm start  

## Built With

React  
Redux  
Webpack
Express
Axios - Used to make AJAX calls to Amazon API
Redux-promise - Middleware
